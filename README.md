 # Unity 2D 셰이더 데모

 ## 프로젝트 개요
 이 프로젝트는 Unity의 Universal Render Pipeline(URP) 환경에서 2D 셰이더 효과를 구현하고 테스트하기 위해 제작되었습니다.
 플레이어 캐릭터를 조작하며 다양한 시각적 효과를 확인할 수 있는 간단한 데모 씬을 포함하고 있습니다.

 ## 주요 기능 및 구현 내용

 ### 1. 기본 플레이어 컨트롤러 (`SimpleController.cs`)
 -   **이동**: 키보드 좌/우 화살표 또는 'A'/'D' 키를 사용하여 캐릭터를 수평으로 움직일 수 있습니다.
 -   **점프**: 스페이스바를 누르면 캐릭터가 점프합니다.
 -   **방향 전환**: 캐릭터의 이동 방향에 따라 스프라이트가 좌우로 자연스럽게 뒤집힙니다.

 ### 2. 셰이더 효과
 이 프로젝트에는 다음과 같은 셰이더 그래프를 활용한 시각 효과가 포함되어 있습니다.

 -   **피격 효과 (`HitEffect.shadergraph`)**
     -   마우스 왼쪽 버튼을 클릭하면 캐릭터가 붉게 점멸하는 피격 효과가 나타납니다.
     -   `HitEffect.cs` 스크립트가 `DOTween`을 사용하여 셰이더의 `_HitEffectAmount` 속성 값을 부드럽게 조절하여 애니메이션을 구현합니다.

 -   **가짜 광원 효과 (`FakeLight.shadergraph`)**
     -   2D 환경에 동적인 광원이 없는 경우를 대비하여, 특정 위치에 빛이 있는 것처럼 보이게 하는 효과입니다.

 -   **물결 효과 (`Wave.shadergraph`)**
     -   텍스처의 UV를 시간에 따라 조작하여 배경 등이 물결처럼 일렁이는 애니메이션 효과를 만듭니다.

 ## 실행 및 테스트 방법

 1.  **프로젝트 열기**:
     -   Unity Hub에서 이 프로젝트 폴더를 추가하고 `Unity 6000.0.60f1` 버전으로 엽니다.
 2.  **씬 실행**:
     -   `Assets/Scenes/SampleScene.unity` 씬을 엽니다.
     -   Unity 에디터 상단의 ▶ (플레이) 버튼을 눌러 씬을 실행합니다.
 3.  **조작**:
     -   **이동/점프**: 키보드 방향키와 스페이스바를 사용합니다.
     -   **피격 효과 테스트**: 게임 화면에서 마우스 왼쪽 버튼을 클릭합니다.

 ## 기술 스택
 -   **엔진**: Unity 6000.0.60f1
 -   **렌더 파이프라인**: Universal Render Pipeline (URP)
 -   **주요 스크립트**: `SimpleController.cs`, `HitEffect.cs`
 -   **셰이더**: `HitEffect.shadergraph`, `FakeLight.shadergraph`, `Wave.shadergraph`
 -   **외부 라이브러리**: DOTween (에셋에 포함)
